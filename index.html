Describtion
With this app, you can log in with google or create a new account, and enjoy with two sections, one for the entertainment and the other for the tools:

Entertainment Section

    Chat App: you can chat with others in real-time and upload photos.

    Connect 3 Game: you can play the connect 3 game with your friends.

    Media Player: you can play and run some music and enjoy watching videos.

    Dog or Cat Quiz: you can run a quiz to detect if you are acting like Dog or cat.

Tools Section

    Calculator: you can do simple operations like Sum, sub, multiplication, and division.

    Temperature Converter: you can convert degrees from or to celsius and Fahrenheit.

    Java T Point website: you surf the java point website and study programming language through it.

    Google Maps: you can get your current location through google map.

    Mobile Track PDF : you can learn the mobile development fundementals.

Features i Used

    App is written solely in the Java programming language.

    Android Studio version 4.1.0

    Gradle version 6.5

    Current Location with Google Maps API.

    Firebase Authentication

    Firebase Database

    Firebase Storage

    Adding UI Widget

    Java language will be used for development in all this designs

    All versions of gradle and android studio are stable.
